Colorado

A TSV/CSV spreadsheet. Colorado allows to visualize and edit CSV and TSV files. It is able to import a file specifying the delimiter character, but the program will infer the format by itself in the 90% of cases.

In order for Colorado to work correctly, it will need Gtk# installed. In GNU/Linux and Mac mono will also be needed.

  • CSV
